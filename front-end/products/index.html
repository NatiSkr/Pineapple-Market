Gestionar inventario

Mediante el acceso a la base de datos, usuarios regulares y administradores podrán:

  • Buscar un producto (por nombre)
  • Ver un listado de productos
  • Ordenar un listado de productos
  • Exportar el inventario como un archivo Excel

Solo los administradorse podrán:

  • Crear un producto
  • Editar un producto
  • Eliminar un producto
